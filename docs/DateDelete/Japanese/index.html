Bucketin アプリ・データの削除方法

アプリの削除方法

1. 設定を開きます。
2. 【アプリと通知】⇒【〇個のアプリをすべて表示】⇒ Bucketinを選択します。
3. 【ストレージとキャッシュ】⇒【ストレージを消去】をタップします。
4. 確認画面が表示するので【OK】をタップします。

データの削除方法

1. 設定を開きます。
2. 【アプリと通知】⇒【〇個のアプリをすべて表示】⇒ Bucketinを選択します。
3. 【アンインストール】をタップします。
4. 確認画面が表示するので【OK】をタップします。
