Bucketin プライバシーポリシー

本プライバシーポリシーは、Crio/Pieceの提供するアプリケーション「Bucketin」により収集、送信されるユーザー情報とその取り扱いについて説明します。
内容をよくご理解の上、本製品をお遊び下さい。

使用ツール

本アプリでは広告配信ツールとして、Unity Technologies社の提供する「Unity Ads」を使用しています。
取得情報、利用目的、第三者への提供等につきましては、広告配信事業者のプライバシーポリシーをご確認下さい。

収集データ：デバイス識別子、広告ID、IPアドレス
