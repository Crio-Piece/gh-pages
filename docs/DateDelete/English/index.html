Bucketin How to delete apps or data

How to delete the app

1. Open the settings.
2. Select [Apps and notifications] ⇒ [Show all 〇 apps] ⇒ Bucketin.
3. Tap [Storage and cache] ⇒ [Clear storage].
4. A confirmation screen will be displayed, so tap [OK].

How to delete data

1. Open the settings.
2. Select [Apps and notifications] ⇒ [Show all 〇 apps] ⇒ Bucketin.
3. Tap [Uninstall].
4. A confirmation screen will be displayed, so tap [OK].
